## Short Project Description

Codex Telegram Bridge is a local Telegram control surface for OpenAI Codex
app-server sessions. It lets developers start Codex from a terminal, leave the
machine, and continue the same active Codex thread from Telegram without
exposing the local app-server to the internet.
